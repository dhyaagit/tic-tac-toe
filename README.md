# Tic-Tac-Toe (C)

A simple two-player Tic-Tac-Toe game built in C for the terminal.

## Features

- Two-player mode (X and O)
- Turn-based gameplay
- Win/draw detection
- Text-based interface

## How to Run

```bash
gcc main.c -o tictactoe
./tictactoe
```
